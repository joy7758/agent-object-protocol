# AOP Enhancement Proposals (AEP)

This directory contains AOP Enhancement Proposals (AEPs): design documents used to propose, discuss, and ratify non-trivial changes to Agent Object Protocol (AOP).

## AEP Index

| AEP | Title | Status | Target | Tracking |
|---:|---|---|---|---|
| 0001 | Versioning and Compatibility Policy for v0.x | Draft | v0.x | (optional) issue link |

## Status Definitions

- Draft: Proposed and under discussion
- Accepted: Approved for inclusion
- Rejected: Not proceeding
- Implemented: Landed in `main` and released/tagged
- Superseded: Replaced by a newer AEP
